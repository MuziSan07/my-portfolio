My Projects

A selection of things I've built: LLM agents, automation bots, transcription systems, and web platforms. Most are open source on GitHub.

WhatsApp AI Assistant

A personal assistant you message on WhatsApp by text or voice note, in English, Hindi, or Hinglish. It books Uber rides, messages contacts, sends reminders, and even calls someone who doesn't reply. Claude handles the reasoning; Whisper transcribes voice; Twilio moves the messages.

Node.js Claude Whisper Twilio Express

Momentum Trading Bot

An automated penny-stock day-trading bot for Alpaca. Each premarket it scans the most active stocks, filters for genuine momentum, runs an AI sanity check, and sizes positions behind hard risk rules like one trade per day and a PDT guard. Ships with CI and runs on paper money by default.

Python Alpaca API Ollama / Claude CI

AI Content Humanizer

A Streamlit app that rewrites AI-generated text into natural, conversational writing while keeping the original meaning intact. A custom system prompt handles tone and clarity; LLaMA 3.1 via the Groq API keeps responses fast.

Python Streamlit Groq LLaMA 3.1

Wabi Sabi GEO Engine

A Streamlit tool for Generative Engine Optimization: it crawls pages with Firecrawl, uses LLMs to rework content so it surfaces well in AI answers, logs results to Google Sheets, and pings IndexNow so search engines pick up changes fast.

Streamlit Firecrawl Groq Anthropic

ATS Resume Generator

Builds ATS-friendly resumes tailored to a specific job posting. It reads the job description, pulls out the keywords that matter, and produces clean TXT and PDF output. Runs on LLaMA 3.3 70B via Groq with LangChain; PDFs are rendered with ReportLab.

Streamlit LangChain Groq ReportLab

HIPAA Medical Chatbot

A medical Q&A chatbot designed to answer health questions without storing personal health information. FastAPI serves a LangChain pipeline running on Groq, with a responsive Bootstrap chat interface on the front end.

FastAPI LangChain Groq Bootstrap

Arabic Transcription Dashboard

A transcription service for Darija (Moroccan Arabic) with a management dashboard on top: token-based usage tracking, rate limiting, and API key management. Built with FastAPI around a Hugging Face Transformers model.

FastAPI Hugging Face Transformers Speech AI

HNS Travels

A live website for a Pakistani travel agency, covering tour packages, destinations, and enquiries. Hand-built with plain HTML, CSS, and JavaScript, and deployed to a custom domain at hnstravel.com.pk.

HTML CSS JavaScript Live

Content Management System

A custom CMS for creating and managing content, built with Django and Django REST Framework. Includes role-based access control, a dynamic editing interface, and a clean admin experience for non-technical users.

Django DRF PostgreSQL Bootstrap

Image Metadata Optimizer

A Streamlit utility for image-heavy freelance work. It strips old EXIF and writes fresh IPTC metadata (title, author, keywords, description, rating) into JPEGs, and batch-renames files with auto-numbering.

Python Streamlit Pillow piexif

Galaxy Morphological Classification

A deep-learning notebook that classifies galaxies by shape and size from telescope imagery. Covers the full workflow from data preparation and augmentation to training and evaluating a convolutional neural network.

Python CNN TensorFlow Jupyter
